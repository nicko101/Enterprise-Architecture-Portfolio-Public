# Cloud Migration & Modernisation

## Overview
This directory serves as the **Implementation Playbook** for the Enterprise Architecture Portfolio. It documents the patterns and governance standards used to modernise legacy infrastructure into Azure-native services.

---

## Architecture Playbooks

###  [Identity & Endpoint Modernisation](./identity-and-endpoint-modernisation/)
* **Focus:** GPO to Intune migration, NDES/SCEP certificate enrollment, and Zero Trust identity.
* **Goal:** Moving the "Identity Perimeter" from on-premises to the cloud.

###  [Network & Transit Security](./network-and-transit-security/)
* **Focus:** Hub-and-Spoke VNet design, Azure Firewall governance, and Hybrid S2S VPN integration.
* **Goal:** Establishing a secure, scalable transport layer for enterprise traffic.

###  [Modern Data Services](./modern-data-services/)
* **Focus:** Azure File Sync, DFS decommissioning, and Private Link implementation.
* **Goal:** Removing public exposure of data and modernising file collaboration.
