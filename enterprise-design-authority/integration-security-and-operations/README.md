# integration, security, and operations

## overview
this directory documents the technical integration of multi-vendor platforms. it defines how the switching fabric, policy engine, and next-gen firewalls operate as a unified zero trust ecosystem.

## architecture suites
* [**palo-alto-security-suite**](./palo-alto-security-suite/) - ngfw standards, traffic inspection, and inter-vr routing.
* [**clearpass-policy-suite**](./clearpass-policy-suite/) - network access control, guest management, and certificate onboarding.
* [**aruba-switching-suite**](./aruba-switching-suite/) - edge enforcement standards and dynamic segmentation.
