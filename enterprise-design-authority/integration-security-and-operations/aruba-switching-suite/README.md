# aruba switching suite
## overview
defines the standards for automated edge security. focuses on dynamic segmentation where switch ports are configured based on the identity of the device/user.

### core pillars
* [**dynamic-user-roles**](./dynamic-user-roles/) - centralized role download via clearpass.
* [**local-user-roles**](./local-user-roles/) - switch-resident role definitions for fallback security.
